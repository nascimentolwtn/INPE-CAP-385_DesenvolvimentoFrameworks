O Esfinge é um framework que possui a filosofía de reduzir ao máximo a dificuldade no desenvolvimento de aplicações corporativas em Java.Ele adota uma filosifia de buscar nos metadados das classes a maior parte das informações necessária. Tudo isso é associado a técnicas que permitem a extensibilidade do framework para sutuações mais especificas e não previstas. Desta forma o Esfinge consegue aliar produtividade com flexibilidade.
